All About Modules

 

 

Modules are boxes that you can place on your HOME or READER template to display content or information. Modules need to be created and, to be displayed, placed into your HOME or READER template.  Each module has settings that can be changed to display information to your liking, but ALL modules have the following settings.

Alignment: This is how the text or content is aligned in your module.  You can choose from Default (reverts to your global setting), Left, Center, Right

 

Available on: You have check boxes for HOME and READER, whichever ones you check will make that module available in the AVAILABLE MODULE menu on that specific page. If you don’t check either, the module will not show up in either.

 

Enter the title of the project as it appears in the URL when viewing the project (leave default value for current project) 

What does this mean? As it says if you want the module to display information pulled from the current project you are working on then leave it as is. If you have multiple projects, and you want the module to display information from a different project then enter its title here. Just make sure you enter it as IT APPEARS IN THE URL.

Remeber when you have your modules in the order that you like within your template, YOU MUST CLICK THE YELLOW SAVE BUTTON. Otherwise any changes you made to the order will be lost.

 

List of Modules

Activity Feed

This module box allows visitors to your HOME or READER page to see all the updates on your project. This is very useful if you have download or gallery pages that may not show obvious evidence of an update upon first glance.

# of Entries to Display: This is the number of updates to display within the module box at any given time.

 

Author’s Comments

This module will display the comments you enter in the “comments” field available when you upload a content piece.

 

Blog

Displays your bog posts in a module box on your HOME or READER pages.

# of Entries to Display: This is the number of updates to display within the module box at any given time

Module Height: you can set the height in pixels

 

Chapter’s List: Displays a List of your chapters. See “Setting up Episodes, Chapters and Modules” tutorial for more information on setting these up.

List Format: vertical thumbs, thumb box, text list

Series Number: If you’re setting chapters for a specific series

Episode Number: If you’re setting chapters in a specific series

 

Comment Form: Allows people to see user comments posted to the comic

Credits: Displays the credits entered under the project’s settings

Custom: Has a form that allows you to enter custom HTML

Downloads: Displays items you have uploaded into the DOWNLOADS content section within the CMS

Episode List: Displays a list of your episodes. See “Setting up Episodes, Chapters and Modules” tutorial for more information on setting these up.

Facebook Like: Adds a module with a Facebook like button.

Forum: Shows the most recent forum post to comic’s forum.

Latest Page: Module box that shows the most recent page published.

Series Number: If you want to display a page from a specific series

Episode Number: If you want to display a page from a specific episode

Links: Displays the links you’ve created in the Links area of the CMS

Mobile: Displays content items you’ve uploaded to the MOBILE section of the CMS

Other Projects: Displays the users other projects

Page Comments: Displays a dialogue box that allows users to enter comments

Series List: Displays a List of your series. See “Setting up Episodes, Chapters and Modules” tutorial for more information on setting these up.

SuperFan Module: A module that displays a link or banner that takes users directly to an interface that allows them to pay and sign up to be a super fan.

Select Banner Type:  Your choice of a banner or box style banner

Synopsis: displays the summary for the comic entered in the setting area of the project
