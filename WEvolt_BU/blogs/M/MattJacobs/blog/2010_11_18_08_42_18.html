But being busy is more just an excuse than anything, so I've gotta get back on the horse and update! I'm going to take a few minutes each week (atleast) to post an update of some kind. It may be a long ramble or a quick snippet, but it will be something!

First up, WEvolt features. I've been able to crank out a bunch of new stuff that has been on the to do list for a while, and I'm really excited to release in the coming weeks.

All the following will be ready soon, some are already in place. And we'll do an announcement with a quick 'How To' for each of them. 

Module Designer
Jason put together a bunch of new module styles and I've created a new wizard that will let you quickly change the look of your boxes. Some have had a hard time customizing modules before, this should make it much easier.

Module Creator
For those that haven't used the new module creator, give it a try on either the 'HOME' content section or READER. You can now create new modules from a list that will be constantly growing. If you have any ideas for modules, speak your mind in the forum here: MODULE MASHUP!

Series Editor
This new feature has been requested by a couple of our long time WEvolt users JamesRiot and Phinmagic. It will add the ability for you to run spin-off / alt series from one project. So with my project Stupid Users, instead of running Spam Kills as its own project. This will now become a new series under Stupid Users, but will be only available to fans. 

Access Control
This feature will let you section off all your project content (even entire content sections) to be seen by either Everyone, Fans or SuperFans.
